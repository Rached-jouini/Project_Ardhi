OU AJOUTER LES SCREENSHOTS SUR LINKEDIN
==================================================

Option recommandee (simple) :
1) Copie-colle le texte final dans le champ de publication LinkedIn.
2) Clique sur l'icone "Photo" / "Media".
3) Ajoute 2 a 4 captures max (qualite claire).
4) Garde l'ordre mentionne dans le texte :
   - Capture 1 = interface principale
   - Capture 2 = fonctionnalite cle
   - Capture 3 = resultat / test

Astuce :
Ajoute une petite ligne avant les hashtags :
"Quelques apercus en images :"
Puis laisse les images attachees au post (pas besoin d'inserer dans le corps du texte).
